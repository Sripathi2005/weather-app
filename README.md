# 🌦️ React Weather App

A sleek and modern **Weather App** built with **React** and the **OpenWeatherMap API**.  
It fetches real-time weather data including **temperature, humidity, wind speed, pressure, visibility, and condition icons**.  

---

## ✨ Features
- 🔍 Search weather by **city name**
- 🌡️ Displays **temperature, feels like, humidity, pressure, wind speed, visibility**
- ⛅ Dynamic **weather condition icons** (sun, clouds, rain, etc.)
- 🎨 Modern **glassmorphism UI** with gradient background
- 📅 Shows **date & time** of data
- ⚡ Built with **React Hooks** (`useState`)

---

## 🚀 Tech Stack
- **React** (Frontend)
- **OpenWeatherMap API** (Weather Data)
- **CSS3** (Glassmorphism & Premium Styling)
